⚙ YouTube publishing setup

One-time, free, ~15 minutes. The Client ID is a public identifier — no secret involved.

⚠ You opened the app from a local file (file://). Google sign-in only works from a real web address — use the GitHub Pages site or a local server (serve.ps1).
  1. Open the Google Cloud Console ↗ and create a project (any name, e.g. TopRankVids).
  2. Enable the YouTube Data API v3 ↗.
  3. In OAuth consent screen ↗: choose External, fill the minimum fields, then under Test users add the Google account(s) that will publish (your own email).
  4. In Credentials ↗: Create Credentials → OAuth client ID → Web application.
